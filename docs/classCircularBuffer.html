Detailed Description

template<typename T>
class CircularBuffer< T >

A Circular Buffer class that is templated to take diverse type arguments The capacity is defined at construction time. Oldest items are culled from the buffer if the buffer gets filled.

Constructor & Destructor Documentation

template<typename T >
CircularBuffer< T >::CircularBuffer ( unsigned int  capacity = TELEMETRYSIZE)

Constructor. Takes a default size parameter as capacity. This is a fixed size capacity (i.e. does not grow). Items added beyond the capacity/size will overwrite the oldest items
