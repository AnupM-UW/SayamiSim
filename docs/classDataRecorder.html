Detailed Description

A Data Recorder class Derived from the Process class and gets scheduled by the Manager class Listens to the Controller and Attitude channel to gather data and write it to a MRU buffer of a specific size. Also will periodically write the buffer content to a file on the disk

Update method. Updates the data in its internal buffer. At a set schedule the Most Recent data points will be flushed to disk as a log file. Called repeatedly by the manager at a frequency determined by the period used when the process is scheduled with the Manager (see Manager::schedule).
